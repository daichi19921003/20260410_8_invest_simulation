# プロダクト要求定義書 (Product Requirements Document)

## プロダクト概要

### 名称
**Gemini Stock Simulator** - Gemini による自動株トレード・シミュレーター (Next.js版)

### プロダクトコンセプト
- **AIによる投資判断**: Gemini (Google Generative AI) を活用し、テクニカルデータ等から「買い・売り・ホールド」を自動判定。
- **モダンな Web アプリケーション**: Next.js、Supabase、shadcn/ui を活用した、高速でインタラクティブな Web UI。
- **データ永続化と同期**: Supabase (PostgreSQL) を利用し、取引履歴やユーザーの資産状況をセキュアに管理。

### プロダクトビジョン
投資家が、最新の AI 技術を活用した自作の投資戦略を、リスクなしで検証・可視化できるプレミアムな Web シミュレーターを提供する。データの信頼性と、直感的かつ美しい UI を兼ね備えた、次世代の投資学習プラットフォームを目指す。

### 目的
- Gemini の投資判断能力の検証と評価。
- リスクなしでの株トレード戦略の学習。
- Next.js と Supabase を組み合わせたモダンな Web アプリケーション開発。

## ターゲットユーザー

### プライマリーペルソナ: 投資に興味があるエンジニア (30歳)
- 属性: Web 開発スキルがあり、AI の活用に関心がある。
- 課題: 自分の投資判断を客観的に検証したい。
- 解決策: AI にデータの提示と判断を行わせることで、データに基づいたトレードを体験する。

## 成功指標(KPI)

### プライマリーKPI
- **シミュレーション実行数**: 累計10回以上のシミュレーション。
- **アプリ満足度**: shadcn/ui による洗練された UI 体験。

## 機能要件

### コア機能(MVP)

#### 1. データ取得・表示
**ユーザーストーリー**:
ユーザーとして、判断のベースとなる株価情報を、リアルタイムに近い形で Web 上で確認したい。
**受け入れ条件**:
- [ ] 外部 API (yfinance 等) から取得したデータを、美しくチャート表示できる。

#### 2. AI 投資判断 (Gemini Integration)
**ユーザーストーリー**:
ユーザーとして、複雑なデータ解析をAIに任せ、売買のアドバイスを得たい。
**受け入れ条件**:
- [ ] サーバーサイド(Server Actions)から Gemini API を呼び出し、判断を取得できる。

#### 3. 取引シミュレーション & 履歴
**ユーザーストーリー**:
ユーザーとして、取引の結果が資産にどう影響したかを履歴で確認したい。
**受け入れ条件**:
- [ ] Supabase への取引記録の保存。
- [ ] 仮想資金(100万円等)の残高計算。

#### 4. ダッシュボード
**ユーザーストーリー**:
ユーザーとして、現在の全体資産を一目で把握したい。
**受け入れ条件**:
- [ ] 資産推移のグラフ表示。

## 非機能要件
- **パフォーマンス**: ページ遷移およびデータ取得の高速化（SSR/ISRの活用）。
- **ユーザー体験**: shadcn/ui によるレスポンシブで洗練されたデザイン。
- **セキュリティ**: Supabase Auth によるユーザー管理（拡張課題）。

## 技術的制約
- フレームワーク: Next.js (App Router)
- データベース: Supabase (PostgreSQL)
- デプロイ: Vercel
